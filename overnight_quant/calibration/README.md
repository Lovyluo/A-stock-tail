# Chip Volume Observation Calibration

This directory stores sanitized calibration summaries for the chip-volume proxy module.

Allowed files:

- `chip_volume_summary_YYYY-MM-DD.md`

Do not commit full live `records/`, full live `reports/`, cache files, logs, or personal trading records. Calibration summaries should contain only aggregate counts and anonymized metric distributions.

The chip and volume fields are proxy indicators for observation confidence. They are not real holder-cost data, not investment advice, and not a standalone buy/sell basis.
