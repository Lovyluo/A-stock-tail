# Disclaimer

A-stock-tail is a research and observation workspace for A-share market data and example overnight watchlist workflows.

This project is not investment advice. Reports, dashboard views, watchlists, scores, backtests, sell plans, and any other generated outputs are for research, learning, and manual review only.

The project does not guarantee profits, trading accuracy, data completeness, data timeliness, or any investment outcome. Market data sources may be delayed, unavailable, incomplete, blocked, revised, or incorrect.

## No Automated Trading

A-stock-tail does not:

- place orders automatically
- click or control brokerage software
- call brokerage trading APIs
- bypass brokerage controls
- manage real money accounts

Any real trading decision must be made manually by the user outside this project.

## Risk Boundary

Use this project only if you understand that:

- backtests are research artifacts, not proof of future performance
- daily proxy data cannot reproduce true intraday decision states
- live data can fail or disconnect
- strategy outputs can be wrong
- you are responsible for all investment decisions and losses

If a feature appears to imply automatic trading, treat that as out of scope for this project and report it as a boundary issue.
